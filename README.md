# MyRealmsAI V0.3 Skeleton

This is a stripped fork of V0.2 that keeps only the application shell:

- `apps/web`: React/Vite UI skeleton with the familiar three-pane layout
- `apps/api`: Express API stub with placeholder endpoints

What is intentionally missing:

- engine
- runtime
- content packages
- LLM calls
- session streaming
- game logic

The goal is to preserve structure and visual language while giving you a clean place to rebuild from.
